### Resumen de carrera

Soy un Desarrollador de Software de Argentina {'🇦🇷🧉'}. He pasado la mayor parte de mi carrera construyendo sistemas backend con Java y Spring Boot, trabajando en entornos fintech y enterprise en América Latina.

Acá está un cronograma rápido:

- **Investigador Académico en Universidad Nacional de San Luis** (Jun 2020 - Jun 2022): Comencé en academia haciendo trabajo de conciencia de ciberseguridad, mentoreando estudiantes y tutoreando admisiones.
- **Desarrollador Apache Groovy en runaID** (Sep 2021 - Jun 2022): Mantuve servicios públicos y sistemas legacy para el gobierno municipal de San Luis usando Groovy Server Pages, jQuery y Apache Tomcat.
- **Desarrollador Next.js en runaID** (Jul 2022 - Oct 2022): Construí un dashboard de auditoría desde cero para procesar y verificar grandes volúmenes de recibos médicos, usando Next.js, React y TypeScript.
- **Desarrollador Java Spring Boot en Applica Solutions** (Nov 2022 - Apr 2023): Estandaricé APIs REST backend para el primer sistema de pagos basado en QR de Argentina.
- **Desarrollador Java Spring Boot en Devsu** (May 2023 - Oct 2025): Integré proveedores de identidad con páginas y aplicaciones orientadas al cliente para el banco privado más grande de Ecuador, trabajando con Google Cloud y Microsoft Azure.
- **Desarrollador de Software (IA/Análisis de Código) en Devsu** (Nov 2025 - Presente): Desarrollando una nueva solución enfocada en analizar bases de código masivas, usando TypeScript, React, Deno, Supabase y RabbitMQ.

Tengo certificaciones en Google Cloud (Professional Cloud Architect, Associate Cloud Engineer), Microsoft Azure (DevOps Engineer Expert, Azure Developer Associate), Scrum (PSM I), y fundamentos de Confluent Kafka, entre otros.

### ¿Por qué "Pollito"?

Quizás te estés preguntando por qué esto es la **Opinión del Pollito** sobre Desarrollo con Spring Boot.

"Pollito" es mi apodo desde mi primer cumpleaños, cuando mi madre me vistió de amarillo brillante. Mis familiares inmediatamente empezaron a llamarme "pollito" y el nombre se quedó.

A lo largo de los años, he adoptado este apodo ya que es original y memorable. En el mundo tech, donde el personal branding importa, tener una identidad distintiva ha sido sorprendentemente útil. Además, ¿quién no sonríe cuando escucha sobre un developer llamado "pollito" construyendo sistemas enterprise para un banco importante?

El logo del pollito y la marca "pollito" se han convertido en mi firma profesional: un recordatorio de que incluso el trabajo serio puede mantener un toque de personalidad y calidez.
